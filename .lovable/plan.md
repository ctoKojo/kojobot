

# تطبيق تحسينات برومبت Kojo

## التغيير الوحيد

**ملف**: `supabase/functions/chat-with-kojo/index.ts` — تعديل `SYSTEM_PROMPT` (سطور 13-34)

### البرومبت الجديد هيشمل:

1. **قاعدة الاعتراف بالإجابة الصح فوراً** — لو الطالب جاوب صح، امدحه وكمّل للخطوة الجاية بدل ما تسأل نفس السؤال تاني.

2. **تعريف "شبه مكتمل" بمثالين**:
   - `number + 2 % 5 == 0` → ترتيب عمليات غلط، المفروض `(number + 2) % 5`
   - `if (x % 2 = 0)` → استخدم `=` بدل `==`
   - **القاعدة**: لو الخطأ الوحيد أقواس أو precedence → امدح + صحّح في سطر واحد مع السبب

3. **أقصى 2 سؤال توجيهي على نفس النقطة** — لو الطالب مش فاهم بعد سؤالين، ادّيه الإجابة واشرحها

4. **قاعدة "مش عارف" مرتين** → ادّيه الإجابة مباشرة

5. **ملخص الحل النهائي** — لما الطالب يخلّص، اجمع الكود كامل في بلوك

6. **اختبار سريع بعد الملخص** — سطر واحد: مدخل → مخرج متوقع (مثال: `لو i=3 → 3+2=5 → 5%5=0 ✓ هيتطبع`)

7. **السؤال في الآخر بس لو لسه فيه خطوات** — لو الحل خلص، لخّص واسأل "عايز نجرب حاجة تانية؟"

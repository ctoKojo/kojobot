## الحلول المقترحة

### 1. تبسيط الحقول (حقل واحد بدلاً من اثنين)

**الفكرة:** حقل نص واحد للسؤال + حقل واحد لكل خيار

```text
┌─────────────────────────────────────────────────────────────────────┐
│                    التصميم الجديد                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  نص السؤال:                                                         │
│  [ما هو الـ loop في البرمجة؟ / What is a loop?________________]    │
│                                                                     │
│  ○ [تكرار كود معين / Repeating code_________________________]      │
│  ○ [متغير / A variable_____________________________________]       │
│  ○ [دالة / A function______________________________________]       │
│  ○ [شرط / A condition______________________________________]       │
│                                                                     │
│  ✅ 5 حقول فقط بدلاً من 10!                                        │
│  💡 المستخدم يكتب باللغة المناسبة (أو كلتيهما)                      │
└─────────────────────────────────────────────────────────────────────┘
```

**ملاحظة مهمة:** لن نحذف الأعمدة القديمة (question_text_ar, options.ar) - سنحتفظ بها للتوافق مع البيانات الموجودة، لكن سنجعل واجهة الإدخال أبسط.

## التغييرات التقنية المطلوبة

### 1. تحديث قاعدة البيانات (Migration)

```sql
-- إضافة عمود للصورة
ALTER TABLE public.quiz_questions 
ADD COLUMN image_url text DEFAULT NULL;
```

### 2. إنشاء Storage Bucket للصور

```sql
-- Bucket للصور
INSERT INTO storage.buckets (id, name, public) 
VALUES ('quiz-images', 'quiz-images', true);

-- سياسة الرفع للمشرفين
CREATE POLICY "Admins can upload quiz images"
ON storage.objects FOR INSERT TO authenticated
WITH CHECK (
  bucket_id = 'quiz-images' 
  AND has_role(auth.uid(), 'admin')
);
```

### 3. الملفات المتأثرة

| الملف | التغيير |
|-------|---------|
| `src/pages/QuizEditor.tsx` | تبسيط الحقول + إضافة رفع الصور + استيراد Excel |

## ملخص الخطة

1. **تبسيط الحقول:** حقل واحد للسؤال + حقل واحد لكل خيار (بدلاً من حقلين لكل منهما)
2. **دعم الصور:** رفع صورة للسؤال (مفيد جداً لأسئلة الكود والرسومات)
3. **استيراد Excel:** قالب Excel للإدخال السريع + رفع وتحويل تلقائي
4. **تحسينات إضافية:** نسخ الأسئلة، معاينة للطالب

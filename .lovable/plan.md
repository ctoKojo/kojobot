
# نظام Kojobot لإدارة التعليم والتدريب - الخطة الكاملة

## نظرة عامة
نظام إدارة تعليمي متكامل بثلاثة أنواع مستخدمين (مدير - انستراكتور - طالب) مع دعم كامل للعربية والإنجليزية، وهوية بصرية مميزة بألوان Kojobot المتدرجة (#61BAE2 → #6455F0).

---

## المرحلة الأولى: البنية الأساسية والمصادقة

### 1. الهوية البصرية وثيم Kojobot
- تطبيق التدرج اللوني (#61BAE2 → #6455F0) كهوية رئيسية
- تصميم عصري ونظيف
- دعم كامل للـ RTL (اتجاه العربية)
- زر تبديل اللغة (عربي/إنجليزي) في كل الصفحات

### 2. نظام المصادقة
- صفحة تسجيل دخول موحدة بتصميم Kojobot
- توجيه تلقائي حسب نوع المستخدم
- نظام أدوار آمن (admin, instructor, student)

---

## المرحلة الثانية: لوحة تحكم المدير (صلاحيات كاملة)

### 1. Dashboard رئيسي
- إحصائيات شاملة: عدد الطلاب، المجموعات، الانستراكتورز
- نظرة سريعة على آخر النشاطات
- تنبيهات الاشتراكات المنتهية

### 2. إدارة الفئات العمرية والمستويات
- **إضافة/تعديل/حذف فئات عمرية** (مثال: 6-9، 10-13، 14-18)
- **إدارة المستويات التعليمية**:
  - Level 0 → Level 1 (مستويات أساسية)
  - Level 2 Software / Level 2 Hardware (تفرعات)
- ربط المستويات بالفئات العمرية

### 3. إدارة الحسابات
- **إنشاء حسابات الطلاب**:
  - البيانات الشخصية والصورة
  - الفئة العمرية والمستوى
  - بيانات الاشتراك (مدة، مبلغ مدفوع، متبقي، تاريخ التجديد)
- **إنشاء حسابات الانستراكتورز**:
  - البيانات الشخصية والصورة
  - التخصصات

### 4. إدارة المجموعات
- إنشاء مجموعات جديدة
- تحديد: الموعد، المدة، الفئة العمرية، المستوى
- إضافة طلاب للمجموعات
- تعيين انستراكتور مسؤول

### 5. بنك الأسئلة
- إنشاء كويزات جاهزة
- تصنيف حسب المستوى والموضوع
- تخزين للاستخدام لاحقاً

### 6. الإعدادات العامة
- تحكم كامل في إعدادات النظام
- إدارة الفئات والمستويات

---

## المرحلة الثالثة: لوحة تحكم الانستراكتور

### 1. Dashboard شخصي
- الصورة والبيانات الشخصية
- عدد المجموعات ومواعيدها
- المهام القادمة

### 2. إدارة المجموعات (المسؤول عنها فقط)
- عرض تفاصيل المجموعة
- قائمة الطلاب
- مواعيد السيشنات

### 3. تسجيل الحضور والغياب
- تسجيل حضور/غياب لكل سيشن
- تتبع تاريخ الحضور

### 4. الكويزات والاساينمنتات
- **إسناد كويز** من بنك الأسئلة لمجموعة أو طالب محدد
- **إنشاء اساينمنت** (نص، صورة، PDF، فيديو) لمجموعة أو طالب
- تحديد مدة التسليم
- متابعة التسليمات والنتائج

### 5. متابعة الطلاب
- عرض بيانات كل طالب
- الحضور والغياب والإنذارات
- الكويزات والاساينمنتات

---

## المرحلة الرابعة: لوحة تحكم الطالب

### 1. Dashboard شخصي
- الصورة والبيانات الشخصية
- السن والفئة العمرية والمستوى
- مدة الاشتراك وتاريخ التجديد

### 2. بيانات المجموعة
- اسم المجموعة والموعد والمدة
- اسم الانستراكتور

### 3. إحصائيات الحضور
- عدد السيشنات المحضورة
- عدد السيشنات الغائبة
- عدد الإنذارات

### 4. الكويزات والاساينمنتات
- عرض الكويزات المسندة (للمجموعة أو له شخصياً)
- حل الكويزات
- عرض الاساينمنتات ورفع التسليمات (نص، صورة، PDF، فيديو)

---

## المرحلة الخامسة: Activity Log (سجل النشاطات)

### تسجيل شامل لكل الأحداث:
- **تسجيل الدخول والخروج** لكل المستخدمين
- **التغييرات على البيانات** (إضافة، تعديل، حذف)
- **الكويزات والاساينمنتات** (إسناد، تسليم، نتائج)
- **الحضور والغياب** (كل تسجيل)
- **مين عمل إيه ومتى وعلى إيه**

### عرض السجل:
- فلترة حسب نوع الحدث
- فلترة حسب المستخدم
- فلترة حسب التاريخ
- تصدير السجل

---

## المرحلة السادسة: نظام الإشعارات (Notifications)

### إشعارات الطلاب:
- كويز جديد مسند
- اساينمنت جديد
- تذكير بموعد التسليم
- نتيجة كويز
- إنذار جديد

### إشعارات الانستراكتور:
- طالب سلم اساينمنت
- طالب غائب
- تحديثات المجموعة
- تغييرات من المدير

### إشعارات المدير:
- اشتراك طالب قارب على الانتهاء
- اشتراك منتهي
- تقارير يومية
- تنبيهات النظام

### مركز الإشعارات:
- أيقونة إشعارات في الـ header
- عداد الإشعارات غير المقروءة
- قائمة الإشعارات مع حالة (مقروء/غير مقروء)
- إشعارات داخل الموقع فقط (بدون إيميل)

---

## البنية التقنية

### قاعدة البيانات (Supabase):
- `profiles` - بيانات المستخدمين
- `user_roles` - أدوار المستخدمين (admin, instructor, student)
- `age_groups` - الفئات العمرية (قابلة للتعديل)
- `levels` - المستويات التعليمية (قابلة للتعديل)
- `groups` - المجموعات
- `group_students` - ربط الطلاب بالمجموعات
- `sessions` - السيشنات
- `attendance` - الحضور والغياب
- `quizzes` - بنك الأسئلة
- `quiz_assignments` - إسناد الكويزات
- `quiz_submissions` - تسليمات الكويزات
- `assignments` - الاساينمنتات
- `assignment_submissions` - تسليمات الاساينمنتات
- `warnings` - الإنذارات
- `subscriptions` - الاشتراكات
- `activity_logs` - سجل النشاطات
- `notifications` - الإشعارات

### التخزين (Supabase Storage):
- صور المستخدمين
- ملفات الاساينمنتات (PDF، صور، فيديو)

### الأمان:
- Row Level Security (RLS) على كل الجداول
- صلاحيات محددة لكل نوع مستخدم
- تسجيل كل النشاطات

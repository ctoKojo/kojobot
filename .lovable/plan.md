
# خطة إعادة تنظيم القوائم والصفحات والعناصر

## الوضع الحالي - تحليل المشاكل

### 1. تكرار وظائف الكويزات (المشكلة الأساسية)

| الموقع | الوظيفة | المشكلة |
|--------|---------|---------|
| `/quizzes` (بنك الأسئلة) | إنشاء + تعديل + حذف + **إسناد** | زر إسناد مكرر هنا |
| `/my-instructor-quizzes` (كويزات المجموعات) | **إسناد** + عرض النتائج | نفس زر الإسناد موجود هنا |
| `/quiz-reports` (تقارير الكويزات) | تقارير عامة | معزول منطقياً |

**النتيجة**: الأدمن يقدر يسند نفس الكويز من مكانين مختلفين!

### 2. تصنيف غير منطقي في السايدبار

```text
القائمة الرئيسية:
  - الداشبورد ✓
  - الطلاب ✓
  - المدربين ✓
  - المجموعات ✓
  - السيشنات ✓
  - جدول العمل (instructor فقط) ✓

التعليم:
  - بنك الأسئلة (admin)
  - كويزات المجموعات (admin + instructor)  ← تكرار في الوظائف
  - كويزاتي (student)
  - تقارير الكويزات (admin)
  - الواجبات (الكل)
  - الحضور (الكل)  ← منطقياً مرتبط بالسيشنات!

الإعدادات:
  - الفئات العمرية
  - المستويات
  - سجل النشاط
  - الإشعارات
  - الإعدادات
```

### 3. أيقونات متكررة

- `FileQuestion` مستخدمة في **4 صفحات** مختلفة (بنك الأسئلة، كويزات المجموعات، كويزاتي، تقارير)
- صعب التفريق بينهم بصرياً

### 4. تسميات غير واضحة

| الاسم الحالي | المشكلة |
|--------------|---------|
| "كويزات المجموعات" | غامض - هل هي كويزات مسندة ولا متاحة؟ |
| "بنك الأسئلة" | الاسم صح لكن موجود فيه وظيفة إسناد ملهاش علاقة |

## الفوائد المتوقعة

1. **وضوح الوظائف**: كل صفحة لها مسؤولية واحدة
2. **تجربة مستخدم أفضل**: لا تكرار في الوظائف
3. **سهولة الصيانة**: منطق الإسناد في مكان واحد فقط
4. **تنظيم منطقي**: الحضور مع السيشنات، الكويزات معاً
5. **تمييز بصري**: أيقونات مختلفة لكل صفحة

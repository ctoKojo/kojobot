

# تحسين اللاندينج بيدج - سيكشنات جديدة وتحسينات

## التغييرات المطلوبة

### 1. تحديث بيانات الباقات في قاعدة البيانات
- تحديث الـ 3 باقات الموجودة بالأسماء الصحيحة: **KOJO SQUAD**, **KOJO CORE**, **KOJO X**
- كل الباقات: 4 حصص/شهر، 120 دقيقة/حصة
- اضافة مميزات كل باقة من الـ PDF كـ `landing_plan_benefits`

**مميزات الباقات من الـ PDF:**

| الميزة | KOJO SQUAD | KOJO CORE | KOJO X |
|--------|-----------|-----------|--------|
| فيديو ملخص بعد كل حصة | لا | نعم | نعم |
| فيديو شرح كامل للسيشن | لا | لا | نعم |
| دعم في الواجبات | لا | فوري ومباشر | فوري ومباشر |
| تواصل مباشر مع المدرب | لا | نعم | نعم |
| حساب ولي الأمر | لا | متابعة شهرية | تقرير + أسبوعي |
| مرونة في المواعيد | لا | لا | نعم |
| SLIDES ومواد تعليمية | لا | نعم | نعم |
| PDF FILES اضافية | لا | لا | نعم |
| أولوية في الجدول | لا | لا | نعم |
| شهادة معتمدة | لا | لا | نعم |
| جروب صغير | نعم | نعم | لا (فردي) |
| دعم يومي على واتساب | لا | لا | نعم |
| تحليل وأداء أسبوعي | لا | لا | نعم |
| جدول ذكي حساب الأسرة | لا | لا | نعم |
| فيديوهات مراجعة للمذاكرة | لا | لا | أسبوعيا |
| تجربة تعليم فردية مخصصة | لا | لا | نعم |
| تقييم أداء فردي | لا | لا | نعم |

### 2. اضافة سيكشن الأسئلة الشائعة (FAQ)
- أسئلة وأجوبة ثابتة (hardcoded) بالعربي والإنجليزي
- تصميم Accordion باستخدام المكون الموجود `@radix-ui/react-accordion`
- أسئلة مثل: ما هو كوجوبوت؟ ما الأعمار المناسبة؟ كيف تتم الحصص؟ هل يوجد تجربة مجانية؟

### 3. اضافة سيكشن تواصل معنا (Contact Us)
- عرض بيانات الاتصال من CMS (ايميل، تليفون، واتساب، عنوان)
- نموذج بسيط او روابط مباشرة للتواصل
- أيقونات واضحة مع تأثيرات hover

### 4. تحسين الخطوط والأبعاد
- التأكد من استخدام `Poppins` للإنجليزي و `Cairo` للعربي في كل العناصر
- تحسين الـ padding والـ spacing بين الأقسام
- ضبط أحجام النصوص والأزرار لتكون متناسقة

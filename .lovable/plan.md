

# خطة إنشاء صفحة جدول عمل المدرب (Instructor Schedule)

## نظرة عامة
إنشاء صفحة جدول عمل أسبوعي لكل مدرب تعرض:
- أيام الأسبوع وساعات العمل المحددة من المدير
- أيام الإجازة مميزة بلون مختلف
- المجموعات المسندة للمدرب تُعرض تلقائياً في الفترات الزمنية المناسبة

## المميزات الرئيسية

### للمدرب (Instructor View)
- عرض جدوله الأسبوعي بشكل مرئي (شبيه بالصورة المرجعية)
- رؤية أيام العمل وساعاتها
- رؤية المجموعات المسندة إليه في كل فترة زمنية
- رؤية أيام الإجازة

### للمدير (Admin View)
- تحديد أيام العمل لكل مدرب
- تحديد ساعات العمل (وقت البداية والنهاية) لكل يوم
- تحديد أيام الإجازة
- المجموعات تُعرض تلقائياً بناءً على المجموعات المسندة للمدرب

## التفاصيل التقنية

### 1. قاعدة البيانات - جدول جديد

سيتم إنشاء جدول `instructor_schedules` لتخزين جدول العمل:

```text
+----------------------+-----------+----------------------------------+
| Column               | Type      | Description                      |
+----------------------+-----------+----------------------------------+
| id                   | uuid      | Primary key                      |
| instructor_id        | uuid      | معرف المدرب (FK -> auth.users)  |
| day_of_week          | text      | اليوم (Sunday, Monday, etc)     |
| is_working_day       | boolean   | هل هو يوم عمل؟                  |
| start_time           | time      | وقت بداية العمل                 |
| end_time             | time      | وقت نهاية العمل                 |
| notes                | text      | ملاحظات اختيارية                |
| notes_ar             | text      | ملاحظات بالعربي                 |
| created_at           | timestamp | تاريخ الإنشاء                   |
| updated_at           | timestamp | تاريخ التحديث                   |
+----------------------+-----------+----------------------------------+
```

**سياسات الأمان (RLS):**
- المدير يمكنه إضافة وتعديل وحذف جدول أي مدرب
- المدرب يمكنه فقط عرض جدوله الخاص

### 5. المميزات البصرية

- **ألوان مختلفة للمجموعات** - كل مجموعة بلون مميز
- **تمييز أيام الإجازة** - خلفية رمادية أو مخططة
- **تمييز ساعات العمل** - الفترات المتاحة مميزة عن غيرها
- **Responsive Design** - يعمل على الموبايل والديسكتوب

### 6. تدفق العمل

```text
المدير                                   المدرب
   |                                        |

   |                                        |
   |  2. إسناد مجموعات للمدرب              |
   |     (تُظهر تلقائياً في الجدول)         |
   |                                        |
   |                                        | 3. عرض الجدول
   |                                        |    (للقراءة فقط)
```

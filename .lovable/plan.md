

## الخطة: عرض أزرار الفيديو المقفولة دايماً للأغراض الترويجية

### المشكلة الحالية
الأزرار "فيديو ملخص" و"فيديو كامل" بتظهر بس لما يكون فيه URL محفوظ في المنهج. الطالب اللي مشترك في باقة كوجو سكواد مثلاً مش بيشوف زرار "فيديو كامل" أصلاً لو الفيديو لسه ماترفعش، فمش بيعرف إن في باقة أعلى ممكن تفتحله المحتوى ده.

### الهدف
خلي الأزرار المقفولة تظهر **دايماً** للطلاب اللي مش عندهم صلاحية، بصرف النظر عن وجود الفيديو من عدمه — لتشجيعهم على ترقية الباقة.

### منطق العرض الجديد

**زرار "فيديو ملخص"** (يحتاج Kojo Core أو Kojo X):
- لو الطالب عنده صلاحية وفيه URL → زرار شغال يفتح الفيديو
- لو الطالب عنده صلاحية ومفيش URL → ما يظهرش (مفيش حاجة يعرضها)
- لو الطالب مالوش صلاحية → **زرار مقفول دايماً يظهر** (سواء فيه URL أو لأ) مع tooltip ترويجي

**زرار "فيديو كامل"** (يحتاج Kojo X):
- نفس المنطق، بس tooltip بيقول "متاح لمشتركي كوجو إكس فقط"

### الملفات المتأثرة

1. **`src/pages/MySessions.tsx`** — كروت السيشنات في صفحة "سيشناتي"
2. **`src/pages/SessionDetails.tsx`** — صفحة تفاصيل السيشن

### تفاصيل التعديل

في الملفين، نغير شرط العرض من:
```
{curriculum.summary_video_url && (canView ? <Button/> : <LockedButton/>)}
```
لـ:
```
{canView 
  ? curriculum.summary_video_url && <Button/>  // عنده صلاحية: يظهر بس لو فيه فيديو
  : <LockedButton/>                            // مالوش صلاحية: يظهر دايماً مقفول
}
```

### نص الـ Tooltips الترويجي (محسّن)

- **زرار الملخص المقفول**: "🔒 افتح الفيديوهات الملخصة بترقية باقتك لكوجو كور أو كوجو إكس"
- **زرار الفيديو الكامل المقفول**: "🔒 شاهد شرح السيشن كامل بالاشتراك في باقة كوجو إكس"

### نقاط مهمة
- المدربين والإدارة مش هيتأثروا (هما أصلاً بيشوفوا كل المحتوى).
- الطلاب اللي عندهم صلاحية مش هيشوفوا أي زرار وهمي لو الفيديو ماترفعش.
- الحفاظ على نفس التصميم البصري (أيقونة قفل + opacity مخفض + cursor-not-allowed).

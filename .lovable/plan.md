

# خطة: نظام توقيت الكويز المجدول

## فهم المتطلبات

الطالب يريد نظام توقيت ذكي للكويزات بحيث:
1. **يبدأ العد التنازلي من وقت البداية المحدد** - وليس من لحظة فتح الطالب للكويز
2. **منع الطالب من فتح الكويز قبل الموعد المحدد**
3. **خصم الوقت المنقضي** - لو فتح الطالب الكويز متأخر 5 دقائق، يرى الوقت ناقص 5 دقائق

---

## التغييرات المطلوبة

### 1. تحديث قاعدة البيانات

إضافة عمود جديد لجدول `quiz_assignments`:

```text
┌─────────────────────────────────────────────────────┐
│              quiz_assignments                       │
├─────────────────────────────────────────────────────┤
│ + start_time (timestamp with time zone) - جديد     │
│   due_date (موجود - سيصبح وقت الانتهاء)             │
└─────────────────────────────────────────────────────┘
```

- `start_time`: وقت بداية الكويز (متى يُسمح للطالب بالدخول)
- `due_date`: وقت نهاية الكويز (الموعد النهائي)

---

### 2. تحديث واجهة إسناد الكويز (Quizzes.tsx)

إضافة حقل **وقت البداية** في نافذة إسناد الكويز:

```text
┌────────────────────────────────────────┐
│         إسناد الكويز                   │
├────────────────────────────────────────┤
│ المجموعة: [اختر مجموعة ▼]             │
│                                        │
│ 🕐 وقت البداية: [____/____/____]       │  ← جديد
│                                        │
│ ⏰ الموعد النهائي: [____/____/____]     │
│                                        │
│         [إلغاء]  [إسناد الكويز]        │
└────────────────────────────────────────┘
```

---

### 3. تحديث صفحة "كويزاتي" (MyQuizzes.tsx)

**منع فتح الكويز قبل موعده:**

```text
┌─────────────────────────────────────────────────┐
│ 📝 كويز الرياضيات                               │
├─────────────────────────────────────────────────┤
│ ⏱️ 30 دقيقة    📅 ينتهي: 15 فبراير            │
│                                                 │
│ ⏳ يبدأ في: الساعة 10:00 صباحاً                │  ← عرض وقت البداية
│                                                 │
│ [🔒 الكويز لم يبدأ بعد] (زر معطل)              │  ← قبل الموعد
│         أو                                      │
│ [▶️ ابدأ الكويز] (زر فعال)                     │  ← بعد الموعد
└─────────────────────────────────────────────────┘
```

---

### 4. تحديث صفحة حل الكويز (TakeQuiz.tsx)

**حساب الوقت المتبقي من وقت البداية:**

```text
المنطق الجديد:
─────────────────────────────────────────────────────
الوقت المتبقي = مدة الكويز - (الوقت الحالي - وقت البداية)

مثال:
• مدة الكويز: 30 دقيقة
• وقت البداية: 10:00 صباحاً
• الطالب فتح الكويز: 10:05 صباحاً (متأخر 5 دقائق)
• الوقت المتبقي = 30 - 5 = 25 دقيقة ✅
─────────────────────────────────────────────────────
```

**التحقق عند فتح الكويز:**
1. إذا `الوقت الحالي < start_time` ← رسالة: "الكويز لم يبدأ بعد"
2. إذا `الوقت الحالي > due_date` ← رسالة: "انتهى وقت الكويز"
3. حساب الوقت المتبقي وبدء العد التنازلي

---

### 5. تحديث Edge Function (grade-quiz)

إضافة تحقق من الوقت قبل قبول الإجابات:

```text
عند استلام الإجابات:
1. تحقق: هل الكويز بدأ؟ (start_time <= now)
2. تحقق: هل الكويز انتهى؟ (now <= due_date + مهلة صغيرة)
3. إذا كل شيء صحيح ← قبول الإجابات وتصحيحها
```

## ملخص التغييرات التقنية

| الملف | التغيير |
|-------|---------|
| **Migration** | إضافة عمود `start_time` لـ `quiz_assignments` |
| **Quizzes.tsx** | إضافة حقل وقت البداية في نافذة الإسناد |
| **MyQuizzes.tsx** | عرض وقت البداية + تعطيل الزر قبل الموعد |
| **TakeQuiz.tsx** | حساب الوقت من `start_time` + منع الدخول المبكر |
| **grade-quiz/index.ts** | التحقق من الوقت قبل قبول الإجابات |

## سيناريوهات الاستخدام

**السيناريو 1: طالب يحاول الدخول مبكراً**
```text
وقت البداية: 10:00 صباحاً
الطالب يحاول الدخول: 9:50 صباحاً
النتيجة: رسالة "الكويز يبدأ الساعة 10:00 صباحاً" + زر معطل
```

**السيناريو 2: طالب يدخل في الوقت**
```text
وقت البداية: 10:00 صباحاً
الطالب يدخل: 10:00 صباحاً
الوقت المتبقي: 30 دقيقة (كاملة)
```

**السيناريو 3: طالب يدخل متأخراً**
```text
وقت البداية: 10:00 صباحاً
الطالب يدخل: 10:15 صباحاً
الوقت المتبقي: 15 دقيقة فقط ⚠️
```

**السيناريو 4: طالب يحاول الدخول بعد انتهاء الوقت**
```text
وقت البداية: 10:00 صباحاً
مدة الكويز: 30 دقيقة
الطالب يحاول الدخول: 10:45 صباحاً
النتيجة: رسالة "انتهى وقت الكويز" + زر معطل
```
